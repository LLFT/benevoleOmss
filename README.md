# benevoleOmss

Il s'agit d'une application permettant de recenser les membres d'une association.
Ces membres sont des bénévoles qui lors de grands évènements enfilent une chasuble réfléchissante de manière à devenir signaleur et assurer l’encadrement.
Ce portail permettra également de définir des parcours ainsi que les spots de signaleur de manière à pouvoir associer un membre volontaire sur un spot défini.
